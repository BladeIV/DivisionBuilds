Division Builds

About

Builds can be submit using the form on the Submit page or via Github Issue or Github Pull Request.
If you want to submt your build via Github Issue or Pull Request you will have to create a Github account first.
After you created your account you can take a look at the Github Repository.
Pull Requests are the fastest way to get your build on the site, but they have to be done properly.
To know how how a build JSON file looks, check out the example.
If you think you can not do a Pull Request with your build data as JSON file properly, no problem. Simply submit a Github Issue or use the form on the Submit page and a member of our staff will turn your build into a JSON file.
Keep in mind that this may take several days or longer, especially if they have to ask you to submit additional information or correct some mistakes
